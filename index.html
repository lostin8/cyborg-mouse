Welcome to 'R.A.T.9 Mouse' Xorg server & evdev configuration for Linux.

About

Due to terrible lack of support from evdev/Xorg in Linux for R.A.T.9 Mouse I decided to write my own evdev settings & configs for it.
Actually there is a few well known problems with this mouse series, starting from:

  • most keys is not binded properly at all;

  • 'horizontal wheel' is not working;

  • short battery lifespan;

  • 'program buttons' are a terrible issue;

  • 'aiming button' is only for decoration.


To read the configuration HowTo press "View on GitHub Button" (top) or follow this link here.
